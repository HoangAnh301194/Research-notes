- Giải thích chi tiết các phương pháp kĩ thuật :
    - Phân loại isolated và continuous Hand gesture recognition 
    - Data colection methods, và đặc điểm bộ dataset của bài báo thu thập

- Kết quả thực nghiện và đánh giá
